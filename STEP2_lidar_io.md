### lidar 가상 환경 활성화하기

윈도우

Anaconda Prompt를 실행하고 아래 커맨드를 입력합니다.
```bash
$ conda activate lidar
```

맥, 리눅스

terminal을 실행하고 아래 커맨드를 입력합니다.
```bash
$ source ~/.bashrc
$ conda activate lidar
```
